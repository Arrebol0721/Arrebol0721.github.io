正则表达式

js 中正则表达式语法规则：

  • / /正则的定义语法
  • ^匹配开始，以后面内容为开始
  • $匹配末尾，以前面内容为末尾
  • 可以通过^()$
  • [0-9][a-z][0-9a-z]指定取值范围
  • [0-9]简写为\d
  • [0-9a-zA-Z_]简写为\w
  • {n}前面内容出现 n 次;{n,}至少出现 n 次;{,m}最多出现 m 次;{n,m}出现 n 到 m 次
  • {0,1}简写为?,表示出现或者不出现
  • {1,}简写为+号
  • {0,}简写为*
