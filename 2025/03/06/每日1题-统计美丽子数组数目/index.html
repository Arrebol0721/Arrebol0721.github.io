每日1题-统计美丽子数组数目

发表于 2025-03-06

题目描述

给你一个下标从 0 开始的整数数组nums 。每次操作中，你可以：

  • 选择两个满足 0 <= i, j < nums.length 的不同下标 i 和 j 。
  • 选择一个非负整数 k ，满足 nums[i] 和 nums[j] 在二进制下的第 k 位（下标编号从 0 开始）是 1 。
  • 将 nums[i] 和 nums[j] 都减去 2的k次方 。

如果一个子数组内执行上述操作若干次后，该子数组可以变成一个全为 0 的数组，那么我们称它是一个美丽的子数组。
请你返回数组 nums 中美丽子数组的数目。

题解

例如{4,3,1,2,4}:
100
011
001
010
100
每一轮操作可以消去某一列两个1，能把所有数变成0，那么这个子数组就是美丽子数组。
我们结合异或的性质，不同为1，相同为0，如果这几个数异或值为0，那么就是美丽子数组。
可以利用前辍和还有哈希表来计算一共有多少连续子数组。
注意：
设置哈希表默认值为{0,1}是在查找前面元素的前辍和的时候。前零个元素的前辍和也应该算上。
